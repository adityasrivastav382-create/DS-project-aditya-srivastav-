# 📊 Data Analysis & Machine Learning Project

## 📌 Overview
This project focuses on performing data preprocessing, analysis, visualization, and machine learning on a real-world dataset. The goal is to extract meaningful insights and build a predictive model.

---

## 🗂️ Dataset
- The dataset contains multiple numerical features related to activity trends.
- It includes missing values and requires preprocessing.

---

## ⚙️ Steps Performed

### 1. Data Preparation
- Removed duplicate records
- Handled missing values using mean
- Checked data types and structure

### 2. Data Analysis
- Calculated statistical measures (mean, min, max, etc.)
- Identified trends and patterns in the dataset

### 3. Data Visualization
- Histogram (distribution of data)
- Box Plot (outlier detection)
- Scatter Plot (relationship between variables)
- Heatmap (correlation analysis)
- Pair Plot (feature relationships)

### 4. Machine Learning
- Applied **Linear Regression**
- Split data into training and testing sets
- Evaluated model using Mean Squared Error (MSE)

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 📈 Results
- Identified key patterns and trends in the dataset
- Built a predictive model to estimate target values
- Visualizations helped in better understanding of data
